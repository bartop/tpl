Detailed Description

template<class Enumerable, class FilterPredicate>
class tpl::filtered_sequence< Enumerable, FilterPredicate >

Sequence filtering given sequence using supplied function.

This class can be safely used with infinite sequences.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.

Template Parameters
EnumerableType of sequence which is to be filtered. Must satisfy is_enumerable trait.
FilterPredicateFunction-like type which will be used for filtering. It should take one argument convertible to Enumerable::value_type and return value of type convertible to bool.
