Detailed Description

template<class Enumerable>
class tpl::keys_sequence< Enumerable >

Sequence extracting keys from input associative sequence.

This class can be safely used with infinite sequences.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.

Template Parameters
EnumerableType of sequence from which keys are to be extracted. Must satisfy is_enumerable and is_associative trait.
