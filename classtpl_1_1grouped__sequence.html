Detailed Description

template<class Enumerable, class Grouping>
class tpl::grouped_sequence< Enumerable, Grouping >

Sequence grouping input sequence into associative container of sequences using given predicate.

This class cannnot be safely used with infinite sequences - it eagerly processes the input sequence.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.

Template Parameters
EnumerableType of sequence which is to be grouped. Must satisfy is_enumerable trait.
GroupingType of function used to group elements from enumerable. It must take one argument constructible from Enumerable::value_type and return any copy-constructible type.
