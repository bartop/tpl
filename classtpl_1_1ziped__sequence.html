Detailed Description

template<class Enumerable1, class Enumerable2>
class tpl::ziped_sequence< Enumerable1, Enumerable2 >

Sequence zipping together two sequences.

This class can be safely used with infinite sequences.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.
