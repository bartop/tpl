Detailed Description

template<class Enumerable>
class tpl::reversing_sequence< Enumerable >

Sequence reversing order of elements in input sequence.

This class cannot be safely used with infinite sequences.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.

Template Parameters
EnumerableType of sequence which is to be reversed.
