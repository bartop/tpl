Detailed Description

template<class Enumerable>
class tpl::cycle_sequence< Enumerable >

Infinite sequence generating repeating sequence of elements. E.g. given sequence { 1, 2 } the output sequence would be { 1, 2, 1, 2, 1,... }.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.

Template Parameters
EnumerableType of sequence to be repeated. Must satisfy is_enumerable trait.
