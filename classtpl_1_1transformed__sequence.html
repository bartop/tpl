Detailed Description

template<class Enumerable, class Predicate>
class tpl::transformed_sequence< Enumerable, Predicate >

Sequence transforming values from input sequence using given predicate.

This class can be safely used with infinite sequences.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.

Template Parameters
EnumerableType of sequence from which elements are to be transformed. Must satisfy is_enumerable trait.
PredicateType of function-like object used to transform elements in input sequence. It must take one argument constructible from Enumerable::value_type and return type which is at least move constructible.
