Detailed Description

template<class Enumerable>
class tpl::taken_sequence< Enumerable >

Sequence taking given number of elements from input sequence.

This class can be safely used with infinite sequences. Actually, it is designed to make finite sequences from infinite ones.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.

Template Parameters
EnumerableType of sequence from which elements are to be taken. Must satisfy is_enumerable trait.
