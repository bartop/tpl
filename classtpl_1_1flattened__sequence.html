Detailed Description

template<class Enumerable>
class tpl::flattened_sequence< Enumerable >

Sequence concatenating internal sequences of input sequence.

This class can be safely used with infinite sequences.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.

Template Parameters
EnumerableType of sequence which is to be flattened. Must satisfy is_enumerable trait. Also Enumerable::value_type must satisfy is_enumerable trait.
