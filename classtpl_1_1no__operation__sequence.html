Detailed Description

template<class Enumerable>
class tpl::no_operation_sequence< Enumerable >

Sequence only wrapping given sequence.

This class can be used as form of abstraction to reduce size of interface of wrapped concrete sequence.

This class can be safely used with infinite sequences.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.
