Detailed Description

template<class Enumerable>
class tpl::cached_sequence< Enumerable >

Sequence caching any other sequence.

This sequence is useful when using other operator which does any kind of heavy or costly processing. It eagerly processes the iterators of given sequence and stores the results so that they are available immidiately. This class due to its nature CANNOT be used with infinite sequences.

This class complies with is_enumerable trait, which allows it to be used in a pipeline.

Template Parameters
EnumerableType of sequence to be cached. Must satisfy is_enumerable trait.
